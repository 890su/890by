# IE Portals Setup — надёжные узлы

Скрипт добавляет белорусские госпорталы в список надёжных узлов (Trusted Sites) и отключает требование HTTPS для этой зоны.

## Что делает скрипт

1. **Добавляет домены в надёжные узлы** — ФСЗН, МНС, ЭСЧФ, Белстат, НЦЭУ
2. **Отключает требование HTTPS** для зоны надёжных узлов

## Добавляемые домены

- `*.vat.gov.by`, `*.nalog.gov.by` — МНС, ЭСЧФ
- `*.ssf.gov.by` — ФСЗН
- `*.belstat.gov.by`, `*.nces.by` — Белстат, НЦЭУ

## Быстрый запуск (одна команда)

Откройте **PowerShell** (права администратора не требуются):

```powershell
irm https://raw.githubusercontent.com/890su/890by/main/ie-portals-setup/ie-portals-setup.ps1 | iex
```

## Запуск через .bat

**Из скачанного архива:**

1. Скачайте репозиторий (Code → Download ZIP)
2. Распакуйте
3. Откройте папку `ie-portals-setup`
4. Запустите **run.bat** двойным щелчком

**Онлайн (актуальная версия):** запустите **run-online.bat** — скрипт скачается с GitHub и выполнится.

## Требования
